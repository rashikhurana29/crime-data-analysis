Crime Data Analysis – Women Safety Metrics
Tools: Python · Pandas · NumPy · Matplotlib · Seaborn
Overview:
Performed end-to-end exploratory data analysis on women safety crime data across major Indian cities (2018–2023). Identified high-risk zones, seasonal patterns, and peak crime hours using Python.
Key Findings:

4,28,278 total incidents analyzed across Pan India
12 high-risk zones identified including Delhi NCR and Mumbai
61% of incidents occur between 10pm – 4am
Harassment accounts for 34.2% of all reported cases

Skills Demonstrated:
Data Cleaning · EDA · Data Visualization · Pattern Analysis · Statistical Analysis
